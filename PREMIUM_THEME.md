# Premium Modern Theme - 设计说明

## 主题概述

这是一个全新的高端、专业、现代化的主题设计,采用优雅的金色/琥珀色为主色调,搭配精致的青色和紫色作为辅助色,营造出奢华、专业且具有视觉冲击力的效果。

## 动画效果

### 过渡时间
- **快速**: 150ms cubic-bezier(0.4, 0, 0.2, 1)
- **正常**: 250ms cubic-bezier(0.4, 0, 0.2, 1)
- **慢速**: 350ms cubic-bezier(0.4, 0, 0.2, 1)
- **平滑**: 400ms cubic-bezier(0.4, 0, 0.2, 1)

### 关键动画
- **淡入向上**: 从下方淡入并向上移动
- **浮动**: 上下轻微浮动
- **脉冲**: 缩放和透明度变化
- **弹跳**: 上下弹跳效果

- **金色球体**: 右上角,500x500px,缓慢浮动

## 无障碍设计

### 焦点状态
- 可见焦点轮廓,金色边框
- 焦点偏移,不遮挡内容

### 减少动画
- 支持 `prefers-reduced-motion`
- 禁用所有动画和过渡
- 隐藏背景装饰元素

### 高对比度模式
- 支持 `prefers-contrast: high`
- 增加边框粗细
- 提高文本对比度

### 屏幕阅读器
- 跳转到主要内容链接
- ARIA 标签和描述
- 语义化 HTML 结构

## 性能优化

### GPU 加速
- 使用 `transform: translateZ(0)`
- `will-change` 属性
- `backface-visibility: hidden`

### 图片优化
- `image-rendering: -webkit-optimize-contrast`
- 适当的图片尺寸

### 字体显示
- `font-display: swap`
- 防止 FOUT (Flash of Unstyled Text)

## 打印样式

- 隐藏背景装饰元素
- 白色背景,黑色文字
- 简化卡片样式
- 避免分页断行

## 设计理念

1. **优雅奢华**: 金色主色调传达专业和高端感
2. **现代科技**: 青色和紫色增添现代感和科技感
3. **层次分明**: 通过颜色、阴影和间距创建清晰层次
4. **视觉平衡**: 每个元素都有其视觉重量和重要性
5. **流畅交互**: 所有动画都经过精心调校,流畅自然
6. **无障碍优先**: 确保所有用户都能访问和使用
7. **性能至上**: 优化渲染和动画性能

## 使用建议

1. **保持一致性**: 在整个项目中使用相同的颜色和间距
2. **适度使用光晕**: 光晕效果很强大,但不要过度使用
3. **注意对比度**: 确保文本和背景有足够的对比度
4. **测试响应式**: 在不同设备上测试布局
5. **考虑性能**: 复杂动画会影响性能,谨慎使用
6. **无障碍测试**: 使用屏幕阅读器测试无障碍性
